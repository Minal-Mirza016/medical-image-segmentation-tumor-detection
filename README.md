# Medical Image Segmentation for Tumor Detection Using MATLAB

A MATLAB-based application that performs medical image segmentation to identify tumor regions using Digital Image Processing techniques. The project demonstrates image preprocessing, enhancement, segmentation, and visualization methods commonly used in medical image analysis.

---

## Project Overview

Medical image segmentation plays an important role in assisting doctors by highlighting abnormal regions in medical scans. This project applies Digital Image Processing techniques in MATLAB to process medical images, enhance their quality, segment the tumor region, and display the final results.

The project was developed as part of a Digital Image Processing course to demonstrate practical implementation of image processing concepts.

---

## Features

- Medical image preprocessing
- Image enhancement
- Noise reduction
- Tumor region segmentation
- Visualization of processed images
- MATLAB-based implementation

---

## Technologies Used

- MATLAB
- Digital Image Processing (DIP)
