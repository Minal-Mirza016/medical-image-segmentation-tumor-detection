# medical-image-segmentation-tumor-detection
A MATLAB-based medical image segmentation system for tumor detection using digital image processing techniques.

## Overview
This project implements medical image segmentation techniques in MATLAB to detect and analyze tumor regions from medical images. The system applies image preprocessing, segmentation, and analysis methods to improve visualization and identification of affected areas.

## Features
- Image preprocessing
- Noise reduction and enhancement
- Tumor region segmentation
- Image analysis and visualization
- MATLAB implementation

## Technologies Used
- MATLAB
- Digital Image Processing

## Project Structure
- main.m : Main MATLAB program
- Report.pdf : Project documentation
